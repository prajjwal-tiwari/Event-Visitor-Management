# Event Visitor Management System

## Overview

QR based visitor registration and verification platform.

## Features

- Visitor Registration
- QR Generation
- QR Verification
- Attendance Tracking
- Dashboard Analytics
- Bulk CSV Upload
- Attendance Export

## Technology Stack

- HTML
- JavaScript
- Supabase
- Netlify

## Business Impact

Reduced manual registration effort and enabled real-time visitor tracking.
